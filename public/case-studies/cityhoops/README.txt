CityHoops case study images
============================

Save the screenshots into THIS folder using these exact filenames.
They are referenced by src/pages/CityHoops.jsx. Until a file exists,
the page shows a labeled placeholder in its spot (so nothing breaks).

  old-home.png      -> old cityhoopsleague.com homepage ("SELECT LEAGUE")
  old-schedule.png  -> old schedule (the flat image / sunday-group.html grid)
  new-home.png      -> new hero ("Play Ball. Track Stats. Love the Game.")
  new-schedule.png  -> new interactive season schedule (week-by-week tables)
  new-stats.png     -> new player stats leaderboard

PNG or JPG both work, but keep the filename + .png extension as written
(or update the extensions in src/pages/CityHoops.jsx to match).
Anything in /public is served from the site root, so these load at
/case-studies/cityhoops/<filename> with no import needed.
