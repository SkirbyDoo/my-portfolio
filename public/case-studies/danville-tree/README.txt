Danville Tree Service case study images
=======================================

Save the screenshots into THIS folder using these exact filenames.
They are referenced by src/pages/DanvilleTree.jsx. Until a file exists,
the page shows a labeled placeholder in its spot (so nothing breaks).

  old-home.png      -> old danvilletreeservice.com homepage (dated, cluttered)
  new-home.png      -> new homepage ("Certified Arborists in the East Bay")
  old-services.png  -> old services section (repetitive blocks / stock photos)
  new-services.png  -> new services section (clean, scannable cards)
  new-mobile.png    -> new site on a phone (with the tap-to-call button)

Tip: capture the old site at https://www.danvilletreeservice.com/ and the
new site at https://danville-tree-service.vercel.app/.

PNG or JPG both work, but keep the filename + .png extension as written
(or update the extensions in src/pages/DanvilleTree.jsx to match).
Anything in /public is served from the site root, so these load at
/case-studies/danville-tree/<filename> with no import needed.

You can also upload these from the dashboard (/admin → Case Studies →
Danville Tree) instead of dropping files here — uploaded images win.
